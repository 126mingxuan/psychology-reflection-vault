# Psychology Reflection Vault

**Sprachen:** [English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja.md) | [Español](./README.es.md) | [Français](./README.fr.md) | [Deutsch](./README.de.md) | [한국어](./README.ko.md) | [Português](./README.pt-BR.md) | [Русский](./README.ru.md) | [العربية](./README.ar.md)

Ein Obsidian-ähnliches Template für einen privaten, lokal orientierten und KI-gestützten Reflexions-Vault.

## Was macht dieses Projekt besonders?

- **Local-first Datenschutz**: Markdown-Dateien bleiben in deinem eigenen Vault. Es gibt keinen Projekt-Server, keine versteckte Datenbank und keine eingebaute Telemetrie.
- **Adaptive Reflexionsstrategien**: Der Assistent kann je nach Antwort zwischen psychodynamischer Exploration, kognitiv-verhaltenstherapeutischen Werkzeugen, Familiensystem-Perspektive, Achtsamkeit, existenzieller Reflexion und Sicherheitsgrenzen wechseln.
- **Langfristige Erinnerung ohne Black Box**: Sitzungen, Fallformulierung und psychologisches Profil sind sichtbare, editierbare Dateien.
- **Öffentliches Template, privates Leben**: Dieses Repository ist öffentlich; echte persönliche Reflexionen gehören in einen privaten Vault.

## Quick Start

1. Klicke auf **Use this template** oder forke das Repository.
2. Erstelle deinen eigenen privaten Arbeits-Vault.
3. Öffne ihn in [Obsidian](https://obsidian.md/) oder einem Markdown-Editor.
4. Starte mit dem Prompt:

```text
Read the core vault files and the latest note in Sessions/.
Continue from the existing psychological reflection system.
Start with one focused opening question.
```

## Sicherheitshinweis

Dieses Projekt ist keine Psychotherapie, Diagnose, psychiatrische Behandlung oder Krisenintervention. Bei akuter Gefahr kontaktiere lokale Notdienste, qualifizierte Fachpersonen oder eine vertraute Person.
