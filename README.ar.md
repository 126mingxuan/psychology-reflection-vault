# Psychology Reflection Vault

**اللغات:** [English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja.md) | [Español](./README.es.md) | [Français](./README.fr.md) | [Deutsch](./README.de.md) | [한국어](./README.ko.md) | [Português](./README.pt-BR.md) | [Русский](./README.ru.md) | [العربية](./README.ar.md)

قالب بأسلوب Obsidian لبناء مساحة خاصة ومحلية أولا للتأمل النفسي المستمر بمساعدة الذكاء الاصطناعي.

## ما الذي يميز هذا المشروع؟

- **خصوصية محلية أولا**: تبقى ملفات Markdown على جهازك أو في مستودعك الخاص. لا يوجد خادم للمشروع، ولا قاعدة بيانات مخفية، ولا تتبع مدمج.
- **استراتيجيات تأمل تكيفية**: يمكن للمساعد أن ينتقل بين الاستكشاف الديناميكي النفسي، وأدوات CBT، ومنظور أنظمة العائلة، واليقظة، والتأمل الوجودي، وحدود السلامة.
- **ذاكرة طويلة المدى بلا صندوق أسود**: ملاحظات الجلسات، وصياغة الحالة المستمرة، والملف النفسي كلها ملفات واضحة وقابلة للتحرير.
- **قالب عام وحياة خاصة**: هذا المستودع عام، لكن التأملات الشخصية الحقيقية يجب أن تبقى في vault خاص.

## Quick Start

1. اضغط **Use this template** أو قم بعمل fork للمستودع.
2. أنشئ vault خاصا لعملك الشخصي.
3. افتحه في [Obsidian](https://obsidian.md/) أو أي محرر Markdown.
4. ابدأ بهذا الطلب:

```text
Read the core vault files and the latest note in Sessions/.
Continue from the existing psychological reflection system.
Start with one focused opening question.
```

## تنبيه سلامة

هذا المشروع ليس علاجا نفسيا أو تشخيصا طبيا أو رعاية نفسية أو تدخلا في الأزمات. إذا كنت في خطر مباشر، فاتصل بخدمات الطوارئ المحلية أو مختص مؤهل أو شخص موثوق.
