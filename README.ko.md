# Psychology Reflection Vault

**언어:** [English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja.md) | [Español](./README.es.md) | [Français](./README.fr.md) | [Deutsch](./README.de.md) | [한국어](./README.ko.md) | [Português](./README.pt-BR.md) | [Русский](./README.ru.md) | [العربية](./README.ar.md)

개인적이고 로컬 우선적인 AI 보조 심리 성찰 vault를 만들기 위한 Obsidian 스타일 템플릿입니다.

## 이 프로젝트의 차별점

- **로컬 우선 프라이버시**: Markdown 파일은 사용자의 기기나 개인 저장소에 남습니다. 프로젝트 서버, 숨겨진 데이터베이스, 내장 텔레메트리가 없습니다.
- **적응형 성찰 전략**: 사용자의 응답에 따라 정신역동적 탐색, CBT 도구, 가족체계 관점, 마음챙김, 실존적 성찰, 안전 경계로 전환할 수 있습니다.
- **블랙박스 없는 장기 기억**: 세션 노트, 진행 중인 사례 이해, 심리 프로필이 모두 읽고 수정할 수 있는 파일입니다.
- **공개 템플릿과 개인 데이터 분리**: 이 저장소는 공개 템플릿이고, 실제 개인 기록은 private vault에 보관해야 합니다.

## Quick Start

1. **Use this template**을 클릭하거나 저장소를 fork합니다.
2. 실제 개인 내용을 저장한다면 작업 vault를 **private**로 유지합니다.
3. [Obsidian](https://obsidian.md/) 또는 Markdown 편집기에서 엽니다.
4. 다음 프롬프트로 시작합니다.

```text
Read the core vault files and the latest note in Sessions/.
Continue from the existing psychological reflection system.
Start with one focused opening question.
```
