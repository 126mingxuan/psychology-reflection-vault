# FAQ

## Is this therapy?

No. This is a self-reflection template. It is not psychotherapy, diagnosis, psychiatric care, or crisis intervention.

## Can I use it with any AI assistant?

Yes. The vault is plain Markdown. You can use it with any assistant that can read or reference files.

## Do I need Obsidian?

No. Obsidian is recommended because it works well with Markdown vaults, but any Markdown editor can work.

## Should my personal vault be public?

No. If it contains real personal material, keep it private or local.

## Is the vault local-first?

Yes. The vault is plain Markdown and has no hosted backend, project server, hidden database, or built-in telemetry.

If you choose to send vault content to a cloud AI service, that service may receive the content you provide. The vault itself is local-first, but your AI provider choice still matters.

## What does adaptive strategy routing mean?

It means the assistant is guided to choose the most useful reflective lens for the current session instead of using one fixed style. Depending on the user's response, it may lean toward psychodynamic exploration, CBT-style tools, family-systems thinking, mindfulness, existential reflection, humanistic support, or safety-first boundaries.

## Why separate session notes from psychological profile?

Session notes hold recent events and emotions. The psychological profile should contain only stable patterns that are repeated or strongly supported.

## Why use one opening question?

One focused question keeps each session clear and prevents the assistant from overwhelming the user with too many directions.

## Can contributors add examples?

Yes, but examples must be fictional and clearly labeled as fictional.
