# Psychology Reflection Vault

**Idiomas:** [English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja.md) | [Español](./README.es.md) | [Français](./README.fr.md) | [Deutsch](./README.de.md) | [한국어](./README.ko.md) | [Português](./README.pt-BR.md) | [Русский](./README.ru.md) | [العربية](./README.ar.md)

Um template no estilo Obsidian para criar um vault privado, local-first e assistido por IA para reflexão psicológica contínua.

## O que torna este projeto diferente?

- **Privacidade local-first**: os arquivos Markdown ficam no seu dispositivo ou no seu repositório privado. Não há servidor do projeto, banco de dados oculto nem telemetria embutida.
- **Estratégias adaptativas de reflexão**: o assistente pode alternar entre exploração psicodinâmica, ferramentas de TCC, sistemas familiares, mindfulness, reflexão existencial e limites de segurança.
- **Memória de longo prazo sem caixa-preta**: notas de sessão, formulação em andamento e perfil psicológico são arquivos visíveis e editáveis.
- **Template público, vida privada**: este repositório é público; suas reflexões reais devem ficar em um vault privado.

## Quick Start

1. Clique em **Use this template** ou faça fork do repositório.
2. Crie seu vault de trabalho privado.
3. Abra em [Obsidian](https://obsidian.md/) ou em qualquer editor Markdown.
4. Comece com este prompt:

```text
Read the core vault files and the latest note in Sessions/.
Continue from the existing psychological reflection system.
Start with one focused opening question.
```

## Aviso de segurança

Este projeto não é terapia, diagnóstico médico, cuidado psiquiátrico ou intervenção de crise. Em caso de perigo imediato, procure serviços de emergência, profissionais qualificados ou uma pessoa de confiança.
