# Psychology Reflection Vault

**Языки:** [English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja.md) | [Español](./README.es.md) | [Français](./README.fr.md) | [Deutsch](./README.de.md) | [한국어](./README.ko.md) | [Português](./README.pt-BR.md) | [Русский](./README.ru.md) | [العربية](./README.ar.md)

Шаблон в стиле Obsidian для создания приватного, локального и AI-assisted пространства психологической рефлексии.

## Чем проект отличается?

- **Local-first приватность**: Markdown-файлы остаются на вашем устройстве или в вашем приватном репозитории. В проекте нет сервера, скрытой базы данных или встроенной телеметрии.
- **Адаптивные стратегии рефлексии**: ассистент может переключаться между психодинамической перспективой, CBT-инструментами, семейными системами, mindfulness, экзистенциальной рефлексией и границами безопасности.
- **Долгосрочная память без черного ящика**: заметки сессий, текущая формулировка и психологический профиль доступны как обычные редактируемые файлы.
- **Публичный шаблон, приватная жизнь**: этот репозиторий открыт, но реальные личные записи должны храниться в private vault.

## Quick Start

1. Нажмите **Use this template** или сделайте fork.
2. Создайте собственный приватный рабочий vault.
3. Откройте его в [Obsidian](https://obsidian.md/) или любом Markdown-редакторе.
4. Начните с этого prompt:

```text
Read the core vault files and the latest note in Sessions/.
Continue from the existing psychological reflection system.
Start with one focused opening question.
```

## Важное предупреждение

Этот проект не является терапией, медицинской диагностикой, психиатрической помощью или кризисной интервенцией. При непосредственной опасности обратитесь в экстренные службы, к квалифицированным специалистам или к человеку, которому доверяете.
